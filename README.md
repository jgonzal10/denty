Denty: Smart Storage System

## Getting Started

First install dependencies:

```bash
npm i
```

Then, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

To run e2e test follow
```bash
npm run dev
npm run e2e
```
